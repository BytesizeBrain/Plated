# ROADMAP FOR BACKEND

### Post-related
`/posts` -> method Post -> Create a new post. Done
`/feed` -> GET all posts. Done
`/posts/<post_id>` -> GET details for one post. Need.
`/posts/<post_id>` -> DELETE a post. Need.

### Like system
Allow users to like/ unlike and see total count

`/posts/<post_id>/like` -> POST -> like a post
`/posts/<post_id>/unlike` -> DELETE -> unlike a post
`posts/<post_id>/likes` -> GET -> Get all likes for a post
